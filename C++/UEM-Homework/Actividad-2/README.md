## UEM – Actividad: Memoria y Punteros en C++

Esta carpeta contiene la documentación correspondiente a la actividad 2 “Memoria (stack y heap) y ejercicios con punteros” de la asignatura Programación con Estructuras Lineales (UEM).

El objetivo de esta práctica es reforzar el entendimiento del modelo de memoria en C++ y afianzar el concepto de punteros mediante ejercicios progresivos.

## Objetivos de la actividad
### Parte 1 — Memoria: Stack vs Heap

El primer bloque consiste en analizar cómo se gestiona la memoria durante la ejecución de un programa:

- Declaración de variables en el stack y el heap.
- Imprimir direcciones de memoria antes y después de modificar valores.
- Comprender la diferencia entre:
    - Dirección del puntero
    - Dirección almacenada dentro del puntero
    - Bloque real reservado en memoria dinámica
- Registrar conclusiones sobre el comportamiento observado.

## Parte 2 — Ejercicios con punteros

La segunda parte del trabajo incluye varios ejercicios destinados a consolidar el uso de punteros en C++.
El alumno debe implementar:

1. Obtener la dirección de memoria de una variable mediante un puntero.
2. Recorrer un array utilizando un puntero e imprimir la dirección de cada elemento.
3. Intercambiar el valor de dos variables empleando punteros.
4. Crear una estructura Estudiante, manipularla mediante un puntero y mostrar su información.

Cada ejercicio debe acompañarse de:
- Comentarios explicativos.
- Uso correcto de sintaxis de punteros.
- Observaciones sobre el comportamiento del programa.
